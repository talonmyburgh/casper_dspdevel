Detailed Description

Function: Signed complex multiply p = a * b when g_conjugate_b = FALSE = (ar + j ai) * (br + j bi) = ar*br - ai*bi + j ( ar*bi + ai*br)

p = a * conj(b) when g_conjugate_b = TRUE = (ar + j ai) * (br - j bi) = ar*br + ai*bi + j (-ar*bi + ai*br) This RTL complex multiply is unlikely to be implemented in DSP48. Use inference version if that is required.
